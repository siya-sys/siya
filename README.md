# SIYA Store — دليل النشر على Vercel

## 🚀 خطوات النشر (5 دقائق فقط)

### الخطوة 1 — إنشاء حساب GitHub
اذهب إلى https://github.com وأنشئ حساباً مجانياً

### الخطوة 2 — رفع المشروع
1. اضغط على زر **"New repository"** الأخضر
2. سمّه: `siya-store`
3. اضغط **"Create repository"**
4. اسحب ملفي `index.html` و `vercel.json` وأرسلهما للصفحة
5. اضغط **"Commit changes"**

### الخطوة 3 — ربط Vercel
1. اذهب إلى https://vercel.com
2. اضغط **"Sign up"** واختر **"Continue with GitHub"**
3. بعد الدخول اضغط **"Add New Project"**
4. اختر repository باسم `siya-store`
5. اضغط **"Deploy"** 🎉

### الخطوة 4 — رابطك جاهز!
بعد 30 ثانية ستحصل على رابطين:
- **رابط المشتري:** `https://siya-store.vercel.app`
- **رابط المدير:** نفس الرابط + اضغط "المدير" في الشريط العلوي

## 🔐 بيانات الدخول للوحة التحكم
- **اسم المستخدم:** `admin`
- **كلمة المرور:** `siya2024`
- يمكنك تغيير كلمة المرور من لوحة التحكم > الإعدادات

## ✨ مميزات المتجر

### للمشتري
- واجهة تقنية أنيقة بألوان أزرق/أبيض/أحمر
- 6 منتجات افتراضية جاهزة
- تصفية حسب الفئة
- سلة تسوق كاملة مع تعديل الكميات
- تصميم متجاوب للجوال والكمبيوتر

### لوحة التحكم (المدير)
- إضافة منتجات جديدة (اسم، سعر، وصف، أيقونة، فئة، لون، شارة)
- حذف المنتجات
- إحصائيات مفصلة بالفئات
- تغيير كلمة المرور

### ملاحظة تقنية
البيانات تُحفظ في `localStorage` — كل مستخدم يرى المنتجات الأصلية.
للحصول على قاعدة بيانات مشتركة يمكن ترقية المتجر لاحقاً بـ Supabase أو Firebase.
